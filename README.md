# Smart Converter

## Overview
Smart Converter is an Android application developed using Kotlin that allows users to convert values between different measurement units. The app is designed with a simple and user-friendly interface, making unit conversions fast, accurate, and easy to perform.

This project was developed as part of an academic assignment to demonstrate Android application development concepts, UI design, and logical implementation using Kotlin.

## Features
- Temperature conversion (Celsius, Fahrenheit, Kelvin)
- Length conversion (Meter, Kilometer, Centimeter, Millimeter, etc.)
- Weight conversion (Kilogram, Gram, Pound, Ounce, etc.)
- Volume conversion (Liter, Milliliter, Cubic Meter, etc.)
- Speed conversion (Meters/second, Kilometers/hour, Miles/hour)
- Area conversion (Square meter, Square kilometer, Acre, Hectare)
- Storage conversion (Byte, Kilobyte, Megabyte, Gigabyte)
- Time conversion (Seconds, Minutes, Hours, Days)

- Splash screen displayed at application launch for improved user experience
- Sliding navigation drawer for intuitive and easy navigation
- Favorites feature to save and quickly access frequently used conversions
- Search bar for fast lookup of conversion categories
- Info / Help section to guide users on application usage

- Settings panel with the following customization options:
  - Dark mode and light mode theme switching
  - Adjustable number of significant figures for conversion results
  - Customizable font size for improved readability

- Clean Material Design-based user interface
- Responsive layout supporting different screen sizes
- Smooth navigation between activities
